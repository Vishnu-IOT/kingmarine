# King Marine Services — Portfolio Website

A premium React portfolio website for King Marine Services, led by Dr. Capt. P.E.J. Rajan.

## Setup & Run

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Build for Production

```bash
npm run build
npm run preview
```

## Pages
- **/** — Home (animated ocean hero, timeline, highlights)
- **/about** — About Us (biography, career, social service)
- **/services** — Services (9 service cards, expansion timeline)
- **/certifications** — Awards & Certifications
- **/contact** — Contact form + information

## Tech Stack
- React 18 + Vite
- React Router DOM v6
- Pure CSS (no UI library)
- Google Fonts: Cinzel, Crimson Pro, Share Tech Mono

## Data
All content is sourced from the JSON files in `src/data/data.js`:
- profile.json
- achievements.json
- business.json
- awards_and_titles.json
- activity.json
- eventachievements.json
